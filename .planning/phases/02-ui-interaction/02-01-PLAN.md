# Plan: 02-01 - Layout & Table [PH2-LAYOUT]

**Goal:** Create the main game layout, the poker table (Tavern themed), and player seats.

## Context
Phase 1 implemented the core engine logic. Now we need a visual space for the game to happen. The "Tavern" theme requires a pixel-art aesthetic with specific colors (Gold, Green, Dark).

## Proposed Changes

### 1. Components
- **`TavernLayout`**: A wrapper component for the game board, ensuring a centered, responsive container.
- **`PokerTable`**: The central visual element. A stylized green-felt table with a gold pixel-art border.
- **`PlayerSeat`**: Displays player information (Name, Chips, Status) and holes cards.
- **`CommunityCards`**: A slot on the table for Flop/Turn/River cards.
- **shadcn/ui Setup**: Initialize required shadcn components: `card`, `button`, `dialog`, `slider`, `badge`.

### 2. Styling
- Use the `pixel-border` utility from `globals.css`.
- Apply `font-pixel` to all text.
- Use `bg-tavern-green` for the table felt.

## Verification Plan

### Automated Tests
- Create `components/__tests__/PokerTable.test.tsx` to ensure components render with correct classes.
- Verify `PlayerSeat` displays chips and names correctly.

### Manual Verification
- Run `npm run dev` and verify the table layout in the browser.
- Check responsiveness (table should scale or scroll on smaller screens).
